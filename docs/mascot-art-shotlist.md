# 噜噜 & 噜妹 出图清单（让表情更丰富）

桌宠的"表情多样"靠多出帧——平面渲染图没法程序化改脸。下面是每个状态建议补的帧。出好图丢进动画包（设置 → 桌面伙伴 → 打开动画包文件夹），按 `manifest.example.json` 配好即可。

## 通用画风提示（保持一致很关键）
- 角色：噜噜（橙黄水豚、头顶橘子）、噜妹（同款、粉色蝴蝶结、腮红）。
- 风格：3D 渲染、柔光、圆润可爱，**正方形构图**、长边 ≤ 512px。
- 背景：**统一**（纯白或纯透明最好——现有两批一批纯白一批淡奶油，混用会让卡片底色不一致）。建议统一成**透明背景**或**纯白**。
- 同一状态的多帧之间，**只改动作幅度、别换场景/机位**（否则循环会跳）。

## 每状态建议帧（state → 帧数 · 动作）

| 状态 | 建议帧 | 动作要点（同机位微变化） |
|---|---|---|
| `idle` | 3 帧 | 噜噜坐着：①平视眨眼 ②微微低头 ③偶尔歪头。轻松待命。 |
| `working` | 3–4 帧 | 噜噜对着屏幕：①打字手左 ②打字手右 ③抬头看屏 ④挠头。专注。 |
| `waiting` | 2–3 帧 | 噜噜看向你/招手：①抬手 ②挥手 ③头顶冒「?」或「!」。要"喊你来"的感觉。 |
| `success` | 3 帧 | 噜噜庆祝：①举手 ②跳起 ③比耶/星星✨。开心。 |
| `error` | 2 帧 | 噜妹/噜噜沮丧：①叹气低头 ②流汗摇头。别太丧，软萌一点。 |
| `sleeping` | 2–3 帧 | 床上睡：①呼吸起 ②呼吸落 ③Z 飘动。 |
| `relax` | 2–3 帧 | 噜妹慵懒：①趴着 ②翻身/伸懒腰 ③喝饮料。松弛感。 |
| `night` | 2 帧 | 桌前打盹：①点头 ②鼻涕泡涨缩。困但还在撑。 |
| `poke`（二期） | 2 帧 | 被戳：①惊一下 ②回弹笑。 |
| `wake`（二期） | 2 帧 | 睡→醒：①猛抬头 ②揉眼。 |

> 已有素材现状：working/success/sleeping/night/relax 各 2 帧；idle/waiting/error 各 1 帧（建议优先补这三个到 2–3 帧）。

## 出图小贴士
- 让模型"出同一角色的连续动作分解帧"，并强调"背景透明、构图机位不变、只改动作"。
- 出完用系统快速操作或在线工具去背（统一透明），再丢进包。
- 帧不必多：2–3 帧循环就比单帧生动很多。
